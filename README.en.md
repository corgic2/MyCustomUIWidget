# CustomCaculator

#### Description
自定义计算器
基本要求
1. 与Windows计算器一致，但仅包含基础四则运算
2. 支持安装程序而非源码获取，及类似应用程序安装
3. 注册至Windows菜单中（安装程序可选）
4. 添加至环境变量，可通过cmd启动（安装程序可选）
5. 支持ini配置项，存在两种及以上布局，包含标准计算器，科学计算器布局，并且会记录上一次关闭的布局选项
